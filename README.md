# Grok Worker

Grok Imagine 전용 이미지 자동화 워커입니다.

이 프로젝트는 `Flow Classic Plus`의 이미지 워커 운영 경험을 그대로 가져오되, 사이트 구조가 다른 Grok Imagine에 맞게 다시 단순화한 별도 프로그램입니다.

## 목표

- 이미지 생성만 담당하는 전용 워커 1개
- 최대 5장의 레퍼런스 이미지를 먼저 업로드하고 재사용
- 프롬프트 안의 `@1`, `@2`, `@3`, `@4`, `@5`를 업로드된 이미지 선택으로 자동 변환
- 생성 후 다운로드, 파일명 변경, 저장까지 자동 처리

## 핵심 아이디어

프롬프트 파일은 Flow 쪽처럼 너무 복잡하게 쓰지 않고, 아래처럼 번호 + 본문 블록으로 단순화하는 방향을 권장합니다.

```txt
001 : @1 @2 cinematic 3d pixar style character...
|||

002 : @1 dramatic lighting, cute 3d render...
|||
```

실행할 때는 각 번호를 `G001`, `G002` 같은 내부 태그로 관리하고, `@1~@5`는 업로드한 이미지 1~5번 선택으로 처리합니다.
