## 현재 방향

- `Flow Classic Plus`의 이미지 워커 운영 지식을 가져오되, Grok 전용 자동화로 새로 분리
- 워커 종류는 하나만 먼저 만든다: `Grok 이미지 워커`
- 동영상/다운로드 전용 워커는 아직 제외

## 핵심 결정

1. 프롬프트 파일 형식은 단순하게 간다.

```txt
001 : @1 cute 3d pixar boy...
|||
```

2. `@1~@5`는 업로드한 레퍼런스 이미지 1~5번을 뜻한다.
3. `@` 선택은 좌표 클릭보다 `아래키 + Enter`로 처리하는 방향이 더 안전하다.
4. 전송은 Enter가 아니라 우측 화살표 버튼 클릭으로 고정한다.
5. 레퍼런스 이미지는 세션 시작 때 먼저 업로드하고, 새로고침 후 재사용한다.
