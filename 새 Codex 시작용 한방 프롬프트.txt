현재 프로젝트는 Flow Classic Plus와 별개인 Grok Imagine 전용 이미지 워커다.

핵심 규칙:
- 워커는 하나만 먼저 만든다: Grok 이미지 워커
- 프롬프트 파일 형식은 `001 : 본문 |||` 기준
- `@1~@5`는 업로드한 이미지 1~5번 선택을 뜻함
- 전송은 우측 화살표 버튼 클릭
- 세션 시작 시 레퍼런스 이미지를 먼저 업로드하고 새로고침 뒤 재사용

지금 상태:
- 기획/설계 문서 작성 완료
- 실제 자동화 코드는 아직 없음
- 다음 구현 우선순위는 UI 골격 -> 설정 저장 -> 프롬프트 파서 -> Playwright 자동화
